# CryptoAlert Bot

Персональный Telegram-бот для приватного отслеживания цен криптовалют с пороговыми и процентными уведомлениями, утренними сводками и режимом тихих часов.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
